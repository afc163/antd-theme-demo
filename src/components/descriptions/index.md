## 何时使用

用于展示一个简单的描述列表

## API

### Descriptions

| 参数       | 说明           | 类型                | 默认值 |
| :--------- | :------------- | :------------------ | :----- |
| title      | 标题           | string              | -      |
| dataSource | 需要展示的信息 | Descriptions.Item[] | -      |

### Descriptions.Item

| 参数    | 说明     | 类型               | 默认值 |
| :------ | :------- | :----------------- | :----- |
| title   | 详情标题 | string             | -      |
| content | 详情信息 | string \| string[] | -      |
